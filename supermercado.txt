RUSUMO SOBRE sistema de controle de validade de produtos do supermercado 

Fui contratado para desenvolver um sistema de vendas para supermercados com um objetivo: ajudar no controle das validades dos alimentos. Garantir que os alimentos estejam dentro do prazo de validade para oferecer mais segurança e qualidade aos consumidores.

Pensando nisso, criei uma solução prática que permite cadastrar produtos com suas informações básicas, como nome, categoria, preço, quantidade em estoque e, especialmente, um dado de validade. Dessa forma, é possível acompanhar quais alimentos estão próximos do vencimento, evitando desperdícios

o sistema também possibilita o gerenciamento dos clientes, facilitando um atendimento mais organizado e personalizado. Para as vendas, o sistema permite registrar as transações rapidamente, selecionando o cliente e os produtos, considerando os itens e atualizando o estoque automaticamente.
Esse sistema foi pensado para funcionar diretamente no navegador, sem complicações de instalação ou servidores externos, usando uma tecnologia de armazenamento local para salvar os dados de forma segura

Requisitos Funcionais

Cadastro de Produtos: Permitir o registro de produtos com nome, categoria, preço, quantidade e data de validade.
Consulta de Validade: Facilitar a busca por produtos com base no nome, categoria ou data de validade.
Gerenciamento de Clientes: Cadastrar e gerenciar informações dos clientes, como nome, e-mail e telefone
Relatórios de Vendas: Gerar relatórios detalhados sobre as vendas
Notificações de Validade: Enviar alertas por e-mail ou SMS sobre produtos próximos do vencimento.

Requisitos Não Funcionais

Usabilidade: Interface intuitiva e fácil de usar.
Acessibilidade: Acesso via navegador, sem necessidade de instalação.
Desempenho: Processamento rápido de consultas e transações.
Segurança: Armazenamento seguro dos dados dos clientes e produtos.
Armazenamento Local: Uso de tecnologia para salvar dados de forma segura entre sessões.
Compatibilidade: Funcionamento em principais navegadores da web.
